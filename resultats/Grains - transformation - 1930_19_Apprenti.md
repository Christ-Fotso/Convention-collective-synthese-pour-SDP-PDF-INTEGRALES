### Rémunération

La convention collective prévoit une rémunération minimale pour les apprentis, calculée en pourcentage du SMIC ou du SMC (Salaire Minimum Conventionnel) selon le niveau et l'échelon de l'apprenti. La convention ne précise pas s'il faut utiliser le SMIC ou le SMC pour la base de calcul.

| Âge        | Année     | % Base   | Base       |
| ----------- | --------- | -------- | ---------- |
| -          | 1ère année |          |            |
| < 18 ans   |           | 25 %     | SMIC/SMC ? |
| 18-20 ans  |           | 37 %     | SMIC/SMC ? |
| 21-25 ans  |           | 53 %     | SMIC/SMC ? |
| > 25 ans   |           | 100 %    | SMIC/SMC ? |
| -          | 2ème année |          |            |
| < 18 ans   |           | 37 %     | SMIC/SMC ? |
| 18-20 ans  |           | 49 %     | SMIC/SMC ? |
| 21-25 ans  |           | 67 %     | SMIC/SMC ? |
| > 25 ans   |           | 100 %    | SMIC/SMC ? |
| -          | 3ème année |          |            |
| < 18 ans   |           | 53 %     | SMIC/SMC ? |
| 18-20 ans  |           | 65 %     | SMIC/SMC ? |
| 21-25 ans  |           | 81 %     | SMIC/SMC ? |
| > 25 ans   |           | 100 %    | SMIC/SMC ? |
