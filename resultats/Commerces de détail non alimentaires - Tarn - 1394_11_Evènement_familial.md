## Congés pour événements familiaux

* **Mariage du salarié :**
    * Durée : 4 jours
    * Conditions : après un an de présence dans l'entreprise
    * Type de jours : ouvrables (les dimanches et jours fériés n'entrent donc pas en compte)
    * La convention ne prévoit rien concernant les modalités de prise, les délais ou les justificatifs.

* **Mariage d'un enfant :**
    * Durée : 1 jour
    * Type de jours : ouvrable
    * La convention ne prévoit rien concernant les modalités de prise, les délais, les conditions d'ancienneté ou les justificatifs.

* **Naissance d'un enfant :**
    * Durée : 3 jours
    * Type de jours : ouvrable
    * La convention ne prévoit rien concernant les modalités de prise, les délais, les conditions d'ancienneté ou les justificatifs.

* **Décès du conjoint, d'un enfant, du père ou de la mère, du beau-père ou de la belle-mère :**
    * Durée : 3 jours
    * Type de jours : ouvrable
    * La convention ne prévoit rien concernant les modalités de prise, les délais, les conditions d'ancienneté ou les justificatifs.

* **Décès d'un ascendant ou d'un descendant, d'un frère, d'une sœur, d'un beau-frère, d'une belle-sœur :**
    * Durée : 1 jour
    * Type de jours : ouvrable
    * La convention ne prévoit rien concernant les modalités de prise, les délais, les conditions d'ancienneté ou les justificatifs.
