### Rémunération des Apprentis

La convention collective prévoit des dispositions spécifiques pour la rémunération des apprentis selon deux cas :

**1. Apprentis non titulaires d'un diplôme du notariat visé à l'article 15.6 de la convention collective nationale du notariat:**

| Diplôme préparé                             | Âge       | Année | % Base | Base |
| ------------------------------------------- | --------- | ----- | ------ | ---- |
| BTS « collaborateur juriste notarial »        | 16-20 ans | -     | 65 %  | E    |
|                                             | 21-25 ans | -     | 75 %  | E    |
|                                             | 26 ans et + | -   | 100 % | SMIC |
| Licence professionnelle métiers du notariat | 16-20 ans | -   | 65 % | E |
|                                             | 21-25 ans | -   | 75 % | E |
|                                             | 26 ans et + | -   | 100% | SMIC |
| DMN                                         | 16-20 ans | - | 65 % | E |
|                                             | 21-25 ans | -     | 75 %  | E    |
|                                             | 26 ans et + | -   | 100 % | SMIC |
| Master droit notarial                       |          |       |        |      |
| DESN                                        |          |       |        |      |



**2. Apprentis titulaires d'un diplôme du notariat visé à l'article 15.6 de la convention collective nationale du notariat et préparant un autre diplôme du notariat :**

| Diplôme obtenu                             | Diplôme préparé                                      | Rémunération                                                                           |
| ------------------------------------------- | ----------------------------------------------------- | ------------------------------------------------------------------------------------- |
| BTS « collaborateur juriste notarial »        | Licence professionnelle métiers du notariat          | 85 % du T1 les 6 premiers mois<br>90 % du T1 les 6 mois suivants                        |
| Licence professionnelle métiers du notariat | DMN ou 1ère année de Master droit notarial            | T1 - 132                                                                              |
| DMN                                         | DMN ou 1ère année de Master droit notarial <br> (niveau atteint pour la 1ère année de Master) | 2ème année de Master droit notarial : 95 % du T2                                        |
| Master droit notarial | DESN   | T2 sans que son coefficient puisse être inférieur la deuxième année à 160 |



**3. Apprentis non titulaires d'un diplôme du notariat visé à l'article 15.6 et préparant un diplôme non prévu par la CCN :**

La convention collective renvoie à l'article D.6222-26 du Code du travail pour la rémunération.

**4. Apprentis titulaires d'un diplôme du notariat visé à l'article 15.6 et préparant un diplôme non prévu par la CCN :**

La convention collective prévoit une rémunération égale à :

| Diplôme obtenu                             | Rémunération                                                              |
| ------------------------------------------- | ------------------------------------------------------------------------ |
| BTS « collaborateur juriste notarial »        | 85 % du T1 les 6 premiers mois puis 90 % du T1                           |
| Licence professionnelle métiers du notariat | T1 - 132                                                                 |
| DMN ou 1ère année de Master droit notarial  | 95 % du T2                                                                |
| Master droit notarial                       | 100 % du T2                                                              |
| DESN | La convention ne prévoit pas de disposition spécifique pour le DESN. |


Pour les salariés de 26 ans et plus, la rémunération applicable est celle prévue par l’article D. 6222-26 4o du Code du travail.

**Autres dispositions :**

- Majoration conventionnelle : La convention prévoit une majoration pour les apprentis titulaires d'une licence professionnelle métiers du notariat (T1-132).
- Prime tutorale : La convention ne prévoit rien à ce sujet.
- Avantages en nature : La convention ne prévoit rien à ce sujet.
- Temps de travail et santé/sécurité : La convention ne prévoit pas de disposition spécifique.
- Dispositions spécifiques régionales : La convention ne prévoit pas de disposition spécifique.
