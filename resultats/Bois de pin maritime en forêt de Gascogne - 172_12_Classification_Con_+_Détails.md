### Classification du Personnel

**Source :** Accord du 10 avril 2002 (Étendu par arrêté du 2 décembre 2002, JO 11 décembre 2002, applicable à compter du 1er janvier 2003)


| Niveau | Échelon | Coeff | Libellé | Statut(s) | Critères | Fonctions |
|---|---|---|---|---|---|---|
| I | O 1 |  |  | Ouvrier | Tâches élémentaires, pas de connaissances particulières requises,  conformité aux consignes, pas d'initiative. |  |
| II | O 2 |  |  | Ouvrier | Tâches simples, pratique ou dextérité acquise, application de procédures préétablies ou données au cas par cas. | |
|  | O 3 |  |  | Ouvrier |  Tâches simples, pratique ou dextérité acquise, peut ordonner ou répartir son travail selon les instructions. |  |
| III | O 4 |  |  | Ouvrier | Tâches combinées, enchaînement de travaux simples selon un mode opératoire détaillé, application de procédures définies et combinées. |  |
|  | O 5 |  |  | Ouvrier | Tâches combinées, enchaînement de travaux simples selon un mode opératoire détaillé, autonome dans les choix des solutions. |  |
| IV | O 6 |  |  | Ouvrier | Travaux complexes, connaissances professionnelles, pratique ou dextérité pour respecter les normes qualité, réglage continu sur machine, définit le mode opératoire. |  |
|  | O 7 |  |  | Ouvrier | Travaux complexes, connaissances professionnelles, pratique ou dextérité pour respecter les normes qualité, réglage continu sur machine, tous domaines de sa spécialité ou effets sur la distribution du travail. |  |
| V | O 8 |  |  | Ouvrier | Tâches, travaux ou actions très complexes, niveau professionnel élevé, connaissance complète de haute valeur technique, propose des modifications de méthodes et procédures. |  |
|  | O 9 |  |  | Ouvrier | Tâches, travaux ou actions très complexes, niveau professionnel élevé, connaissance complète de haute valeur technique, autonomie pour mettre en œuvre méthodes, procédures et moyens, réalise et améliore les objectifs. |  |
| I | ACT 1 |  |  | Administratif, Commercial et Technique | Tâches d'exécution élémentaire, pas de connaissances particulières,  conformité aux consignes, pas d'initiative. |  |
| II | ACT 2 |  |  | Administratif, Commercial et Technique | Tâches d'exécution simple, pratique ou dextérité acquise, application de procédures préétablies ou données au cas par cas. | |
|  | ACT 3 |  |  | Administratif, Commercial et Technique | Tâches d'exécution simple, pratique ou dextérité acquise, peut ordonner ou répartir son travail selon les instructions. |  |
| III | ACT 4 |  |  | Administratif, Commercial et Technique | Tâches diversifiées, ensemble d'éléments ou de consignes administratives ou commerciales, pratique professionnelle ou adaptation des connaissances, met en œuvre des procédures définies et combinées. |  |
|  | ACT 5 |  |  | Administratif, Commercial et Technique |  Tâches diversifiées, ensemble d'éléments ou de consignes administratives ou commerciales, pratique professionnelle ou adaptation des connaissances, met en œuvre des procédures nécessitant réflexion, contrôle et recherche d'informations. |  |
| IV | ACT 6 |  |  | Administratif, Commercial et Technique | Tâches complexes spécifiques, connaissances professionnelles, utilisation de procédures, méthodes, organisation ou technique, analyse et interprète les données pour adapter le mode de réalisation. |  |
|  | ACT 7 |  |  | Administratif, Commercial et Technique | Tâches complexes spécifiques, connaissances professionnelles, utilisation de procédures, méthodes, organisation ou technique, détermine le mode de réalisation, influence la qualité des travaux ou l'efficacité de l'organisation. |  |
| V | ACT 8 |  |  | Administratif, Commercial et Technique | Tâches, travaux ou actions complexes, niveau professionnel élevé, connaissances techniques ou expérience, propose modifications de méthodes, procédures et moyens, améliore les objectifs. |  |
|  | ACT 9 |  |  | Administratif, Commercial et Technique | Tâches, travaux ou actions complexes, niveau professionnel élevé, connaissances techniques ou expérience, autonomie pour mettre en œuvre méthodes, procédures et moyens, réalise les objectifs. |  |
| IV | AM 1 |  |  | Agent de Maîtrise | Assure les responsabilités humaines et techniques, encadrement d'un groupe, organise les travaux et y participe, respecte les consignes, s'assure de la sécurité et de l'application des règles. |  |
|  | AM 2 |  |  | Agent de Maîtrise |  Prend des initiatives, modifications ponctuelles sur l'organisation de l'équipe et les interventions. |  |
| V | AM 3 |  |  | Agent de Maîtrise | Apporte une assistance technique, décide des modifications techniques mineures pour la qualité et les normes. |  |
|  | AM 4 |  |  | Agent de Maîtrise | Participe à l'élaboration de programmes, assiste la direction ou un cadre, assure la mise en œuvre. |  |
| VI | AM 5 |  |  | Agent de Maîtrise | Large autonomie, élabore les programmes d'actions, contrôle la mise en œuvre, objectifs généraux fixés. |  |
| VI | C 1 |  |  | Cadre | Responsable de l'organisation des actions, travaux ou réalisations dans un secteur ou une fonction. |  |
| VII | C 2 |  |  | Cadre | Responsable d'un service ou d'une fonction, coordination d'autres secteurs, activité détermine les actions ou objectifs d'autres services, délégation de pouvoir limitée; ou coordination de plusieurs secteurs, analyse des résultats, participe à l'élaboration des plans généraux. |  |
|  | C 3 |  |  | Cadre | Responsable d'unité de production ou d'un établissement, prérogatives de chef d'établissement; ou élabore et met en œuvre les plans et budgets généraux.  |  |
| VIII | C 4 |  |  | Cadre | Cadre dirigeant, direction intégrale de l'entreprise. |  |


**Modalités d'évolution :** La convention ne prévoit rien à ce sujet.

**Lien avec rémunération :** Les salaires minimums sont liés aux niveaux et catégories. Un point d'ancienneté est prévu, dont la valeur est négociée annuellement.

**Spécificités régionales/départementales :**  RAS.
