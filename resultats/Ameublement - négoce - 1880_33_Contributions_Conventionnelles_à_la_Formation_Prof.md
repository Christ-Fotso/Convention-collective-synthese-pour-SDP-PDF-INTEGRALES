- **Taux CONVENTIONNEL applicable:**  Les entreprises entrant dans le champ d'application de la convention collective du négoce de l'ameublement sont tenues de verser leur contribution à l'OPCO désigné par la branche selon les modalités définies par les articles R. 6332-22-2 et suivants du code du travail. Les taux sont les suivants :

    * Entreprises de moins de 10 salariés : 0,55 % pour le plan de formation, 0,40 % pour le contrat de professionnalisation, 0,15 % pour le CPF.
    * Entreprises de 10 à moins de 50 salariés : 1 % (ou 0,8 % si gestion interne du CPF), 0,20 % pour le plan de formation, 0,30 % pour le contrat de professionnalisation, 0,20 % pour le CPF, 0,15 % pour le CIF, 0,15 % pour le FPSPP.
    * Entreprises de 50 à moins de 300 salariés : 0,10 % pour le plan de formation, 0,30 % pour le contrat de professionnalisation, 0,20 % pour le CPF, 0,20 % pour le CIF, 0,20 % pour le FPSPP.
    * Entreprises de 300 salariés et plus : 0,40 % pour le contrat de professionnalisation, 0,20 % pour le CPF, 0,20 % pour le CIF, 0,20 % pour le FPSPP.

- **Base de calcul spécifique:** Non précisé.

- **Cotisation minimale due (conventionnelle):** Pour les entreprises de moins de 10 salariés, la contribution minimale est de 50 €.

- **Organisme collecteur DÉSIGNÉ par la convention:** AGEFOS PME (remplacé par OPCommerce dans l'accord du 2 octobre 2019).

- **Affectation spécifique des fonds:** La convention ne précise pas d'affectation spécifique des fonds.

- **Entreprises/Salariés concernés:**  Toutes les entreprises entrant dans le champ d'application de la convention collective du négoce de l'ameublement.  Des taux spécifiques sont prévus en fonction de la taille de l'entreprise.

- **Source de l’information:** Article 19 de l'accord du 26 mai 2015 (modifié par l'accord du 2 octobre 2019).
