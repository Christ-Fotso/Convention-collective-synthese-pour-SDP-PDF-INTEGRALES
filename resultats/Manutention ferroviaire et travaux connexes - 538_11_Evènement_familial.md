## Congés pour Événements Familiaux

* **Mariage (de l'intéressé) / PACS :** 4 jours. La convention ne précise pas le type de jours. La convention ne prévoit rien à ce sujet concernant les conditions d'octroi, les modalités de prise ou les jours supplémentaires.
* **Mariage d'un enfant :** 2 jours. La convention ne précise pas le type de jours. La convention ne prévoit rien à ce sujet concernant les conditions d'octroi, les modalités de prise ou les jours supplémentaires.
* **Décès d'un conjoint / partenaire de PACS / concubin / père / mère :** 3 jours. La convention ne précise pas le type de jours. La convention ne prévoit rien à ce sujet concernant les conditions d'octroi, les modalités de prise ou les jours supplémentaires.
* **Décès d'un enfant :** 5 jours. La convention ne précise pas le type de jours. La convention ne prévoit rien à ce sujet concernant les conditions d'octroi, les modalités de prise ou les jours supplémentaires.
* **Naissance d'un enfant / Adoption :** 3 jours. La convention ne précise pas le type de jours. La convention ne prévoit rien à ce sujet concernant les conditions d'octroi, les modalités de prise ou les jours supplémentaires.
* **Décès d'un grand-parent / petit-enfant :** 3 jours. La convention ne précise pas le type de jours. La convention ne prévoit rien à ce sujet concernant les conditions d'octroi, les modalités de prise ou les jours supplémentaires.
* **Décès d'un beau-parent :** 3 jours. La convention ne précise pas le type de jours. La convention ne prévoit rien à ce sujet concernant les conditions d'octroi, les modalités de prise ou les jours supplémentaires.
* **Décès d'un frère / d'une sœur :** 3 jours. La convention ne précise pas le type de jours. La convention ne prévoit rien à ce sujet concernant les conditions d'octroi, les modalités de prise ou les jours supplémentaires.
* **Délais de route (lié à un congé exceptionnel) :** Autorisation d'absence possible.  Peut être non rémunérée ou décomptée dans la limite de 10 jours ouvrables sur le congé annuel payé, le congé d'ancienneté, ou le congé supplémentaire.
