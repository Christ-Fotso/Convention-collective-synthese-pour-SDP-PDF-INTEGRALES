### Classification des cadres

(Se reporter à l’article 14 des dispositions cadres)

| Catégorie | Coefficient |  Libellé      | Statut | Critères                                                                                                                                                                                                                | Fonctions Types                                                                          |
| :-------- | :--------- | :----------- | :----- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------ |
| I         |            | Cadre débutant | Cadre  | Diplômé sans expérience pratique et professionnelle.                                                                                                                                                                      |                                                                                       |
| IA         |            |              | Cadre  |                                                                                                                                                                                                                           |                                                                                       |
| II        |            |              | Cadre  | Cadre technique, administratif ou commercial placé sous les ordres d’un chef de service ou d’un chef de département, a la responsabilité personnelle dans la limite de la délégation qui lui a été consentie. |  Travaux d'ordre technique, administratif ou commercial, sans commandement d'autres cadres. |
| III       |            |              | Cadre  | Grande compétence en gestion commerciale, connaissances technologiques et techniques.                                                                                                                               | Chef de département ou chef de service, avec au moins un cadre sous ses ordres.          |
| IV        |            |              | Cadre  |                                                                                                                                                                                                                           | Cadre dirigeant ou directeur d'un ou plusieurs points de vente, responsabilité de la gestion complète. |

**Modalités d'évolution :** La convention ne prévoit rien à ce sujet.

**Lien avec rémunération :** Les salaires de base de chaque catégorie,  cadres et non-cadres, sont déterminés selon la classification et font l'objet d'accords de salaires.  La convention ne mentionne pas de valeur du point.


**Spécificités régionales/départementales :**  RAS
