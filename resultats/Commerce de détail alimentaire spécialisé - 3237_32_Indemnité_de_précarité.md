Taux conventionnel : 10 % (Article L. 1243-8 du Code du travail).

Réduction du taux : Oui, sous les conditions suivantes (Article L. 1243-10 du Code du travail) :
* CDD conclu pour remplacer un salarié absent ;
* CDD conclu pour exécuter des travaux urgents nécessités par des mesures de sécurité ;
* CDD conclu en attente de l'entrée en service effective d'un salarié recruté par CDI ;
* CDD conclu pour un emploi saisonnier ;
* CDD conclu avec un jeune pour une période comprise dans ses vacances scolaires ou universitaires ;
* CDD conclu dans le cadre d'un accroissement temporaire d'activité.

Cas d'exclusion spécifiques : La convention ne prévoit pas de cas d'exclusion spécifiques concernant l'indemnité de fin de contrat CDD.

Modalités de versement : L'indemnité est versée à la fin du contrat (Article L. 1243-8 du Code du travail).
