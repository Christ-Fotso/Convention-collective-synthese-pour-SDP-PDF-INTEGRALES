## Classification Con + Détails

La convention collective du bâtiment des ouvriers de Haute-Normandie (IDCC 1722) utilise un système de classification avec des niveaux, des coefficients et des libellés, principalement pour définir les salaires minima.  La source principale est l'Avenant n°18 du 20 octobre 2014 (étendu par arrêté du 5 mai 2015, JO 12 juin, applicable à compter du 1er janvier 2015).

| Niveau | Coefficient | Libellé | Statut(s) | Critères | Fonctions |
|---|---|---|---|---|---|
| I - Ouvriers d'exécution | 150 | Position 1 | Ouvrier | La convention ne précise pas les critères de classement pour ce niveau. | La convention ne fournit pas d'exemples de fonctions. |
|  | 170 | Position 2 | Ouvrier |  |  |
| II - Ouvriers Professionnels | 185 |  | Ouvrier |  |  |
| III - Compagnons Professionnels | 210 | Position 1 | Ouvrier |  |  |
|  | 230 | Position 2 | Ouvrier |  |  |
| IV - Maîtres Ouvriers ou Chefs d'Équipe | 250 | Position 1 | Ouvrier |  |  |
|  | 270 | Position 2 | Ouvrier |  |  |


**Modalités d'évolution :** La convention ne prévoit rien à ce sujet.

**Lien avec la rémunération :**  Les salaires minimaux sont liés aux coefficients. Une "valeur de point" est utilisée pour le calcul des salaires pour les coefficients 170, 185, 250 et 270 (voir section "Grille de Rémunération" pour le détail du calcul).  Un "calcul du salaire binôme" est mentionné pour les coefficients 210 et 230 à partir du 1er janvier 2016, mais la convention ne détaille pas ce calcul.

**Spécificités régionales/départementales :** RAS.
